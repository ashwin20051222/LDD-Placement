================================================================================
                         TROUBLESHOOTING
================================================================================

Problem: "gcc: command not found"
Solution: sudo apt install build-essential

Problem: "Kernel headers not found"
Solution: sudo apt install linux-headers-$(uname -r)

Problem: "permission denied"
Solution: Check folder permissions, use cd with correct path

Problem: Module fails to load
Solution: Check with: dmesg | tail -20

Problem: No device /dev/gpioirqdev
Solution: Run: make setup-auto

For more help: See README.md troubleshooting section
