 COMMON COMMANDS:

   make clean              # Clean build files
   make                    # Build the module
   sudo insmod             # Load module
   sudo rmmod              # Unload module
   make setup-auto         # Create device node
   make dmesg-follow       # Follow kernel logs
   ./quickstart.sh         # Interactive setup



 TROUBLESHOOTING:

   Problem: "gcc: command not found"
   Solution: sudo apt install build-essential

   Problem: "Kernel headers not found"
   Solution: sudo apt install linux-headers-$(uname -r)

   Problem: "permission denied" when running make
   Solution: Check folder permissions or cd to correct directory

   Problem: Module fails to load
   Solution: Check with: dmesg | tail -20

   Problem: No device /dev/gpioirqdev
   Solution: Run: make setup-auto

   More help in README.md troubleshooting section!
