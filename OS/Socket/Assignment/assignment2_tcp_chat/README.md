# Assignment 2: TCP Chat Application

## Build

```sh
make
```

## Run

Terminal 1:

```sh
./server
```

Terminal 2:

```sh
./client
```

Type messages on both sides. Type `exit` from either side to stop.

## Sample Output

Server:

```text
TCP chat server listening on port 5002...
Client Connected: 127.0.0.1:40322
[Client]: Hello
[Server]: Hi Client
```

Client:

```text
[Client]: Hello
[Server]: Hi Client
```
