# Assignment 1: Simple TCP Hello Communication

## Build

```sh
make
```

## Run

Terminal 1:

```sh
./server
```

Terminal 2:

```sh
./client
```

## Sample Output

Server:

```text
TCP Hello server listening on port 5001...
Client Connected: 127.0.0.1
Received: Hello Server
```

Client:

```text
Server Reply: Hello Client
```
