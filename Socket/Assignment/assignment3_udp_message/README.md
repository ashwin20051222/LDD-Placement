# Assignment 3: UDP Message Transfer

## Build

```sh
make
```

## Run

Terminal 1:

```sh
./server
```

Terminal 2:

```sh
./client
```

## Sample Output

Client:

```text
Sent: Networking Lab
Received: NETWORKING LAB
```

Server:

```text
UDP server listening on port 5003...
Received from 127.0.0.1:40328: Networking Lab
```
